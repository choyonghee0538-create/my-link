# 사용자 시나리오 정의서 (User Scenarios) - "My Link"

본 시나리오 정의서는 **"My Link"** 서비스의 핵심 주체인 **소유자(Owner)**와 **방문자(Visitor)** 두 그룹의 핵심 행동 흐름을 세부 단계별로 명확히 정의합니다.

---

## 1. [소유자 시나리오] 마이링크 페이지 개설 및 링크 관리 (추가 / 수정 / 삭제)

**목표**: 서비스 소유자는 자신만의 고유 링크 페이지를 개설하고, 홍보할 링크를 자유롭게 추가, 수정, 삭제하여 관리합니다.

### 1단계: 가입 및 나만의 링크 주소 선점
1. **서비스 접속**: 소유자는 "My Link"의 웹사이트에 접속합니다.
2. **고유 주소 신청**: 메인 페이지 검색창에 원하는 고유 이름(예: `miso_beauty`)을 입력하여 중복 검사를 거칩니다.
3. **가입 완료**: 중복되지 않은 주소임을 확인한 후, 이메일과 비밀번호를 입력해 가입을 완료하고 대시보드(관리자 화면)에 진입합니다. 이로써 `mylink.at/miso_beauty`라는 고유 접속 주소가 소유자에게 귀속됩니다.

### 2단계: 프로필 브랜딩 설정
1. **프로필 정보 입력**: 소유자는 퍼블릭 페이지 상단에 들어갈 프로필 아바타 이미지(사진 업로드), 프로필명(`뷰티 크리에이터 미소`), 한 줄 소개(`꿀팁 공유 공간 🌸`)를 입력합니다.
2. **저장 및 프리뷰**: 입력 후 [저장]을 누르면 대시보드 우측의 실시간 모바일 프리뷰 화면에 즉시 소유자의 프로필 정보가 예쁘게 적용됩니다.

### 3단계: 링크 추가 (Add)
1. **새 블록 만들기**: 소유자는 대시보드에서 **[새 링크 추가]** 버튼을 클릭합니다.
2. **정보 입력**: 화면에 생성된 빈 링크 블록에 제목(`💄 5월 단독 특가 공동구매 (~5/25)`)과 연결하고자 하는 외부 URL(`https://smartstore.naver.com/...`)을 입력합니다.
3. **활성화 및 확인**: 블록의 노출 활성화 스위치(기본 ON)를 확인합니다. 대시보드 우측 프리뷰 모니터에 새로 추가한 버튼이 실시간으로 예쁘게 렌더링되는 것을 확인합니다.

### 4단계: 링크 수정 (Modify)
1. **내용 수정**: 소유자는 이미 등록된 유튜브 채널 링크의 오탈자를 발견하거나 제목을 더 매력적으로 변경하고 싶어 합니다. 
   - 제목을 `📺 유튜브 뷰티 튜토리얼`에서 `📺 [신규 영상] 봄 맞이 데일리 메이크업`으로 수정합니다.
2. **순서 정렬 (드래그 앤 드롭)**: 새로 추가한 공구 링크를 가장 위에 배치하기 위해, 공구 링크 블록 카드의 좌측 드래그 핸들을 터치/클릭하여 목록 맨 위로 끌어올립니다.
3. **노출 상태 일시 토글**: 지난주에 종료된 이벤트 링크 블록이 있습니다. 영구히 지우지는 않고 임시로 감추기 위해 블록 우측의 **'활성화 토글'**을 OFF로 전환합니다. 프리뷰 화면에서 해당 버튼이 즉시 사라지는 것을 확인합니다.

### 5단계: 링크 삭제 (Delete)
1. **삭제 기능 실행**: 소유자는 다시 사용할 계획이 전혀 없는 오래된 파트너십 제안 링크를 완전히 제거하기로 결정합니다.
2. **삭제 확정**: 링크 블록 하단에 배치된 **[휴지통 아이콘(삭제)]**을 클릭합니다.
3. **경고 및 완료**: "정말 이 링크를 영구히 삭제하시겠습니까?" 팝업창이 나타나면 [삭제] 버튼을 눌러 확정합니다. 관리 목록 및 대시보드 프리뷰 화면에서 해당 블록이 깔끔하게 완전 삭제됩니다.

## 2. [방문자 시나리오] 마이링크 페이지 방문 및 링크 클릭 이동

**목표**: 소셜 미디어를 서핑하던 방문자는 소유자의 프로필 주소를 타고 들어와 정보를 확인하고 타깃 목적지로 신속하고 안전하게 이동합니다.

### 1단계: 마이링크 페이지 방문 진입
1. **링크 발견**: 방문자는 인스타그램, 유튜브, 틱톡 등을 구경하던 중 소유자의 프로필 바이오 영역에 적힌 링크(`mylink.at/miso_beauty`)를 발견합니다.
2. **터치 및 인입**: 해당 URL 주소를 손가락으로 가볍게 탭합니다.

### 2단계: 모바일 최적화 페이지 로드 및 탐색
1. **초고속 로딩**: 브라우저 새 창이 열리며 "My Link"의 랜딩 페이지가 0.5초 이내에 지연 없이 부드럽게 로드됩니다.
2. **비주얼 확인**: 최상단에 세련된 원형 아바타 프로필 이미지와 소유자가 설정한 한글 닉네임(`뷰티 크리에이터 미소 🌸`), 친절한 한 줄 소개글이 눈에 들어옵니다.
3. **활성화 링크 리스트 탐색**:
   - 프로필 아래로 소유자가 가장 최근에 정렬한 순서대로 세련된 디자인의 링크 버튼들이 차례로 보입니다.
   - 1위로 배치된 `💄 5월 단독 특가 공동구매 (~5/25)` 버튼과 그 아래의 `📺 [신규 영상] 봄 맞이 데일리 메이크업` 버튼이 모바일에 최적화된 큰 탭 영역으로 노출됩니다.
   - (소유자가 비활성화하거나 영구 삭제한 링크들은 방문자 페이지에 전혀 보이지 않아 지저분한 느낌 없이 깔끔합니다.)

### 3단계: 링크 클릭 및 최종 목적지 이동
1. **타깃 클릭**: 방문자는 공동구매 혜택을 받기 위해 `💄 5월 단독 특가 공동구매 (~5/25)` 버튼을 클릭합니다.
2. **인터랙션 반응**: 버튼을 누르는 순간 세련되고 미세한 크기 축소/확대 애니메이션(Micro-interaction)이 반응하여 올바르게 클릭되었음을 시각적으로 알려줍니다.
3. **목적지 도달**: 페이지는 사용자가 설정한 쇼핑몰 상세 페이지(네이버 스마트스토어 등)로 부드럽고 안전하게 리다이렉트되어 방문자가 상품을 즉시 구매할 수 있도록 돕습니다.
