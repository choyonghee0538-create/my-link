# 화면 와이어프레임 설계서 (UI Wireframes) - "My Link"

본 문서는 **"My Link"** 서비스의 2대 핵심 화면인 **1) 소유자용 관리 대시보드(Admin Dashboard)**와 **2) 방문자용 퍼블릭 페이지(Public Profile Page)**의 구조 및 레이아웃을 정의합니다.

---

## 1. [화면 1] 소유자용 관리 대시보드 (Admin Dashboard)

소유자가 프로필을 편집하고 링크 블록을 추가, 수정, 정렬, 삭제하는 메인 작업 환경입니다. 대시보드는 **좌측 설정/편집 영역**과 **우측 실시간 모바일 프리뷰 영역**의 2분할 레이아웃으로 설계됩니다.

### 1.1 ASCII 와이어프레임 레이아웃

```text
============================================================================================
 [My Link] 관리자 대시보드                                           [로그아웃]  [공유하기]
============================================================================================
 [ 템플릿/디자인 ]   [ 링크 블록 관리 ]*   [ 계정 설정 ]
--------------------------------------------------------------------------------------------
 (좌측) 설정 및 편집 패널                         | (우측) 실시간 모바일 프리뷰 (Sticky)
                                                |
 +--------------------------------------------+ |    +----------------------------------+
 |  [1] 프로필 정보 설정                       | |    |          [실시간 프리뷰]         |
 |  ----------------------------------------  | |    |  +----------------------------+  |
 |  아바타 이미지:  [ 이미지 업로드 ] [삭제]   | |    |  |        (  아바타  )        |  |
 |  프로필명:     [ 뷰티 크리에이터 미소 🌸 ] | |    |  |   뷰티 크리에이터 미소 🌸  |  |
 |  소개글 (Bio):  [ 꿀팁 공유 공간 🌸       ] | |    |  |   꿀팁 공유 공간 🌸        |  |
 |                                            | |    |  +----------------------------+  |
 +--------------------------------------------+ |    |  |                            |  |
                                                | |    |  | [ 💄 5월 단독 특가 공구 ]  |  |
 +--------------------------------------------+ |    |  |                            |  |
 |  [2] 링크 블록 목록    [ + 새 링크 추가 ]  | |    |  | [ 📺 봄맞이 메이크업 영상 ]|  |
 |  ----------------------------------------  | |    |  |                            |  |
 |  [::] 카드 1 (활성: ON)                   | |    |  +----------------------------+  |
 |       제목: [ 💄 5월 단독 특가 공구    ]   | |    |                                  |
 |       링크: [ https://smartstore.na... ]   | |    +----------------------------------+
 |       [토글: ON]  [정렬 핸들]  [삭제(휴지통)]| |
 |                                            | |
 |  [::] 카드 2 (활성: ON)                   | |
 |       제목: [ 📺 봄맞이 메이크업 영상  ]   | |
 |       링크: [ https://youtube.com/v... ]   | |
 |       [토글: ON]  [정렬 핸들]  [삭제(휴지통)]| |
 |                                            | |
 |  [::] 카드 3 (비활성: OFF)                 | |
 |       제목: [ ✍ 네이버 블로그 리뷰     ]   | |
 |       링크: [ https://blog.naver.co... ]   | |
 |       [토글: OFF] [정렬 핸들]  [삭제(휴지통)]| |
 +--------------------------------------------+ |
============================================================================================
```

### 1.2 구성 컴포넌트 정보
- **상단 내비게이션 바**: 로고, 로그아웃 버튼, 퍼블릭 URL 주소 복사 및 [공유하기] 단축 버튼.
- **좌측 탭 메뉴**: 프로필 설정, 링크 블록 설정, 계정 설정 전환 버튼.
- **[1] 프로필 설정 컴포넌트**: 아바타 파일 드롭존, 닉네임 입력 폼, 바이오 텍스트 영역.
- **[2] 링크 블록 목록 컴포넌트**:
  - `[+ 새 링크 추가]` 버튼: 클릭 시 목록 가장 위에 빈 링크 카드 추가.
  - `[::]` 드래그 정렬 핸들 아이콘: 드래그 앤 드롭으로 카드 순서 실시간 재배치.
  - `노출 ON/OFF 토글 스위치`: 비활성화 시 프리뷰 및 실제 페이지에서 숨김 처리.
  - `휴지통 아이콘`: 클릭 시 확인 다이얼로그 후 영구 삭제.
- **우측 고정형 프리뷰 모바일 목업**: 화면 스크롤 시에도 우측에 고정되어 실시간 변경 결과를 비주얼로 피드백.

## 2. [화면 2] 방문자용 퍼블릭 페이지 (Visitor Public Page)

사용자가 설정한 주소(`mylink.at/{username}`)로 진입했을 때 모바일/PC 브라우저에 표시되는 최적화 랜딩 페이지입니다.

### 2.1 ASCII 와이어프레임 레이아웃

```text
                     +----------------------------------+
                     |            ( 아바타 )            |  <-- [1] 프로필 이미지
                     |                                  |
                     |       뷰티 크리에이터 미소 🌸    |  <-- [2] 프로필명
                     |        꿀팁 공유 공간 🌸         |  <-- [3] 소개글 (Bio)
                     |                                  |
                     |  +----------------------------+  |
                     |  |  💄 5월 단독 특가 공동구매  |  |  <-- [4] 활성 링크 1 (최상단 강조)
                     |  +----------------------------+  |
                     |                                  |
                     |  +----------------------------+  |
                     |  |  📺 봄맞이 메이크업 영상    |  |  <-- [4] 활성 링크 2 (버튼 스타일)
                     |  +----------------------------+  |
                     |                                  |
                     |                                  |
                     |           [ My Link ]            |  <-- [5] 푸터 브랜드 로고
                     +----------------------------------+
```

### 2.2 구성 컴포넌트 정보
- **[1] 프로필 아바타**: 업로드된 이미지를 둥근 서클로 렌더링. 테두리에 세련된 그라데이션 라인 포인트.
- **[2] 프로필명 / [3] 소개글**: 화면 정중앙에 텍스트 정렬, 가독성이 뛰어난 영문/국문 프리미엄 고딕 서체 사용.
- **[4] 링크 목록 버튼블록 (Link Buttons)**: 
  - 모바일 터치 영역에 부합하도록 넓은 패딩과 둥근 모서리가 적용된 터치형 버튼 스타일.
  - 마우스를 올리거나(Hover) 모바일로 터치(Active) 시 투명도 변화, 미세한 크기 변형 애니메이션을 부여해 생동감 있는 인터랙션 전달.
  - (비활성화 상태인 3번째 블로그 링크는 목록에서 제외되어 표시되지 않음)
- **[5] 푸터 영역**: My Link 브랜드 워터마크 표시.
